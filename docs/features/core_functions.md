# Core Dashboard Functions

## 概要
N-LINE起動直後に表示されるダッシュボード（メイン画面）の機能についての説明です。
プロセスの監視、制御、メンテナンスを行う基本的なツール群です。

## ステータス
- **最終更新日**: 2025-12-14
- **バージョン**: v0.1.0
- **状態**: Stable

## 機能詳細

### 1. Status Monitor
- **内容**: LINEが起動しているか停止しているかを常時監視します。
- **表示**:
    - 🟢 **APP RUNNING**: 起動中
    - 🔴 **STOPPED**: 停止中

### 2. Kill LINE Process
- **内容**: LINEの全プロセス (`LINE.exe`, `LineLauncher.exe` 等) を強制終了します。
- **用途**: フリーズ時の復旧や、完全な再起動を行いたい場合。

### 3. Clear Cache
- **内容**: LINEの一時ファイルを削除し、動作を軽量化します。
- **対象**: `%USERPROFILE%\AppData\Local\LINE\Data\Cache` 配下のファイル。
- **注意**: LINEが起動中の場合は実行できません（ファイルロックのため）。

### 4. Launch LINE
- **内容**: インストールされているLINEを起動します。
- **対応パス**: 標準的なインストール場所 (`AppData\Local\LINE\bin`) を自動探索します。

### 5. Open Install Folder
- **内容**: LINEのインストールディレクトリをエクスプローラーで開きます。
- **用途**: 手動でのファイル確認やModの配置などに便利です。
