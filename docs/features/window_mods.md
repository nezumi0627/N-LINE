# Window Mods

## 概要
LINEウィンドウ自体を操作したり、追加機能をオーバーレイ表示したりする機能群です。

## 機能一覧

### 1. Window Properties
- **Opacity (透明度)**: ウィンドウの不透明度を 30% ~ 100% の間で調整できます。「ながら作業」に最適です。
- **Always on Top (最前面固定)**: LINEウィンドウを他のすべてのウィンドウより手前に表示し続けます。
- **Set Title**: ウィンドウタイトルを一時的に書き換えます（ネタ機能や識別用）。

※ 以前ここに含まれていた **Arg Injection** 機能は、[**Visual Style Injection (QSS)**](qss_injection.md)として独立しました。

## 使用方法

1. **"Window Mods"** タブを開く。
2. **"Find Main Window"** をクリックしてLINEをターゲットにする。
3. **Opacity**: スライダーを動かして透明度を変更。
4. **Topmost**: スイッチをONにして最前面固定。
5. **Set Title**: 新しいウィンドウタイトルを入力して "Set Title" をクリック。
