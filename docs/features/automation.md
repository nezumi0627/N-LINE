# Automation (Experimental)

## 概要
LINEのチャット画面に対して、ユーザー入力のエミュレーションを行う実験的な機能です。
UI Automation APIを利用して、特定クラス（`AutoSuggestTextArea`）を探索し、直接テキストを送信します。

## ステータス
- **最終更新日**: 2025-12-11
- **バージョン**: v0.1.0 (Alpha)
- **状態**: Experimental (動作不安定な場合あり)

## 機能詳細

### 1. Type Text (AutoSuggestTextArea)
- **内容**: チャットの入力欄（テキストボックス）を自動検出し、任意の文字列を入力します。
- **技術**: クリップボードを使わず、`SendKeys` メソッドを使用するため、他の作業を阻害しにくいです。
- **ターゲット**: ClassName `AutoSuggestTextArea` を持つ要素を再帰的に探します。

### 2. Send 'Enter' Key
- **内容**: プログラム的に `Enter` キー押下イベントを入力欄に送信します。
- **用途**: 送信ボタンを押さずにメッセージを送信するために使用します。

## 使用方法

1. LINEでメッセージを送りたい相手とのトーク画面を開いておく。
2. **"Automation"** タブを選択。
3. テキストボックスに送信したい文字を入力。
4. **"Type Text"** をクリック → 入力欄に文字が入る。
5. **"Send 'Enter' Key"** をクリック → 送信される。

## 注意事項
- LINEのバージョンアップにより、クラス名 (`AutoSuggestTextArea`) が変更されると動作しなくなります。
- その場合は **UI Inspector** で新しいクラス名を調べる必要があります。
