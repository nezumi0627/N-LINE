# Window Mods

## Overview
A set of features for manipulating the LINE window itself or displaying additional functional overlays.

## Features List

### 1. Window Properties
- **Opacity**: Adjust the window opacity between 30% and 100%. Ideal for multitasking.
- **Always on Top**: Keeps the LINE window displayed in front of all other windows.
- **Set Title**: Temporarily renames the window title (for fun or identification).

*Note: The **Arg Injection** feature previously included here has been separated into [**Visual Style Injection (QSS)**](qss_injection.md).*

## Usage

1. Open the **"Window Mods"** tab.
2. Click **"Find Main Window"** to target LINE.
3. **Opacity**: Move the slider to change transparency.
4. **Topmost**: Toggle the switch to ON to pin the window to the front.
5. **Set Title**: Enter a new window title and click "Set Title".
