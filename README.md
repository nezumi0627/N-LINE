# N-LINE (Provisional) - Desktop LINE Mod Manager

This application is a utility/mod manager for the Desktop version of LINE.
It allows you to manage the LINE process and perform maintenance tasks.

## Features (v0.1.1)
- **Status Monitor**: Real-time checking if LINE is running.
- **Process Killer**: Forcefully terminate LINE if it freezes or you need to restart it quickly.
- **Cache Cleaner**: Clear LINE's temporary files to free up space and fix loading issues.
- **Mod/Theme Framework**: (Coming Soon) Structure in place to inject custom themes or modify resources.

## How to Run
1. Ensure Python 3 is installed.
2. Run `run.bat`.

## Development
- Built with Python & CustomTkinter.
- Core logic in `core/line_manager.py`.
- UI in `gui/app.py`.
