**N-LINE** は、Windows上のLINEデスクトップアプリケーションを管理、デバッグ、および変更するための強力なPythonベースのユーティリティです。タスク管理（プロセス終了、キャッシュ削除）や、内部UI構造の調査、操作の自動化を行うための高度なエンジニアリングツールを備えた最新のGUIダッシュボードを提供します。

## 🔥 クイックスタート

1. **インストール**: Python 3.11+ がインストールされていることを確認してください。
2. **実行**: `run.bat` をダブルクリックします。
3. **完了**: アプリが依存関係（`customtkinter`, `psutil`, `pywin32` など）を自動的に処理して起動します。
